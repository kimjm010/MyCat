## MyCat

## Description
* 랜덤 고양이 이미지 추천 어플
* 작업 기간: 2022.09.13 - 2022.10.05 (약 0.7개월)

## Environment
* 개발 환경(IDE): Xcode 13 ~ 14 버전
* 개발 언어: Swift
* Target: iOS 14.0 이상
* 소스 관리: Git

## Library
* Alamofire, Gallery, Kingfisher, ProgressHUD

## Implementations
* REST API를 이용항 CRUD기능 구현
* Cat 이미지 목록 페이징 기능
* Cat 이미지 즐겨찾기 추가 및 삭제 기능
* 사용자가 업로드한 Cat 이미지 확인 및 삭제 기능
